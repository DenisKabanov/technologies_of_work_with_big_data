Скачивание образа (начала требуется запустить Docker Desktop в качестве демона (просто с рабочего стола и выполнить вход в аккаунт)):
docker pull bitnami/kafka

Поднятие докера в виде демона (-d):
docker-compose up -d

Отключение докера:
docker-compose down

Запуск:
c:/Users/User/anaconda3/envs/gpu/python generation.py
c:/Users/User/anaconda3/envs/gpu/python processing.py
c:/Users/User/anaconda3/envs/gpu/python ML_inference.py
c:/Users/User/anaconda3/envs/gpu/python -m streamlit run visualization.py
